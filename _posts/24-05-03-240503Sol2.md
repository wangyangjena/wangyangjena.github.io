# P4396 [AHOI2013] 作业



## 思路



把一对 (l\r, a\b) 看成一个点

题意转化为子矩形求和问题



<img src="https://cdn.luogu.com.cn/upload/image_hosting/n2lvrdy6.png" style="zoom:50%;" />

按照 Cdq 分治的流程， 我们需要计算 mid 左侧的点对右侧点的贡献， 即  (x, y) 的左下方一共有多少点



计算上面的内容是简单的， 所以可以把矩形求和拆成几部分， 使得每一部分都是对 (1, 1) $\to$ (x, y) 的矩形求和， 我们把他拆成：（类似二维前缀和）

1. (1, 1) $\to$ (r, b)， operator = +
2. (1, 1) $\to$ (r, a - 1)， operator = -
3. (1, 1) $\to$ (l - 1, b)， operator = -
4. (1, 1) $\to$ (l - 1, a - 1)， operator = +



然后这个问题就变成了一个三维偏序问题， 不等关系如下：


$$
i < j \\
idx_i < idx_j \\
val_i < val_j \\
$$

idx 表示 l/r， val 表示 a/b

然后套板子即可



如何统计 “至少出现过一次的数的个数”：

设 $pre_i$ 表示值为 i 的元素上一次出现的位置

在区间 [l, r] 中， i 是第一次出现， 等价于 $pre_i < l$ 



## 代码



```cpp
#include <bits/stdc++.h> 
#define int long long
using namespace std;
const int LIM = 1e5 + 10;
const int N = 1e6 + 10;

int n, m;
int v[N], l, r, a, b;
int pos[N];

struct Bit{
    int t[LIM];

    int lowbit(int x){
        return x & (-x);
    }

    void Modify(int pos, int x){
        // if(!pos) return;
        while(pos < LIM){
            t[pos] += x;
            pos += lowbit(pos);
        }
    }

    int Query(int pos){
        int ans = 0;
        while(pos){
            ans += t[pos];
            pos -= lowbit(pos);
        }
        return ans;
    }

    void Reset(int x){
        for(int i = 1; i < n; i++){
            t[i] = 0;
        }
    }
}t;

struct Qu{
    int opt, idx, val, op, ansid, pre;

    bool operator < (const Qu &p)const{
        if(val == p.val){
            return opt < p.opt;
        }
        return val < p.val;
    }
}q[N], temp[N];
int cntq, ans[N], anss[N];

bool cmp(Qu x, Qu y){
    if(x.idx == y.idx){
        if(x.opt == y.opt){
            return x.val < y.val;
        }
        return x.opt < y.opt;
    }
    return x.idx < y.idx;
}

void Calc(int l, int r){
    if(l >= r) return;

    int mid = (l + r) / 2;
    Calc(l, mid), Calc(mid + 1, r);

    int i = l, j = mid + 1, cnt = 0, sum = 0;
    // cout << l << " " << r << "LR\n";
    while(i <= mid && j <= r){
        if(q[i] < q[j]){
            if(q[i].opt == 1){
                t.Modify(q[i].pre, 1);
                sum++;
            }
            temp[++cnt] = q[i];
            ++i;
        }else{
            if(q[j].opt == 2){
                ans[q[j].ansid] += q[j].op * sum;
                anss[q[j].ansid] += q[j].op * t.Query(q[j].pre);
            }
            temp[++cnt] = q[j];
            ++j;
        }
    }
    while(i <= mid){
        if(q[i].opt == 1){
            t.Modify(q[i].pre, 1);
            sum++;
        }        
        temp[++cnt] = q[i];
        ++i;        
    }
    while(j <= r){
        if(q[j].opt == 2){
            ans[q[j].ansid] += q[j].op * sum;
            anss[q[j].ansid] += q[j].op * t.Query(q[j].pre);
        }
        temp[++cnt] = q[j];
        ++j;        
    }
    for(int i = l; i <= mid; i++){
        if(q[i].opt == 1){
            t.Modify(q[i].pre, -1);
        }
    }

    for(int i = 1; i <= cnt; i++){
        q[l + i - 1] = temp[i];
    }
}

signed main(){
    // freopen("1.in", "r", stdin);
    // freopen("1.out", "w", stdout);
    ios::sync_with_stdio(0);
    cin.tie(0); cout.tie(0);

    cin >> n >> m;
    for(int i = 1; i <= n; i++){
        cin >> v[i];
        // cout << i << "+" << pos[v[i]] << "\n";
        q[++cntq] = {1, i, v[i], 1, 0, pos[v[i]] + 1};
        pos[v[i]] = i;
    }
    for(int i = 1; i <= m; i++){
        cin >> l >> r >> a >> b;
        q[++cntq] = {2, r, b, 1, i, l};
        q[++cntq] = {2, r, a - 1, -1, i, l};
        q[++cntq] = {2, l - 1, b, -1, i, l};
        q[++cntq] = {2, l - 1, a - 1, 1, i, l};
    }
    sort(q + 1, q + 1 + cntq, cmp);

    Calc(1, cntq);

    for(int i = 1; i <= m; i++){
        cout << ans[i] << " " << anss[i] << "\n";
    }
}
```
